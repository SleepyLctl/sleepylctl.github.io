# CyberPope Site Instructions

1. Unzip this folder.
2. IMPORTANT: Copy your images into the root folder (same place as index.html). 
   You need these images:
   - cyberpope-logo.png
   - petr_trainer_OSCE.jpg
   - petr_pospisil_main.jpg
   - undp-logo.png
   - osce.jpg
   - eeas.png
   - ministry_of_foreign_affrais-cz.png
   - cissp.png
   - securityx.png
   - gcti.png
   - gcda.png

3. VIEWING THE SITE:
   Because this site uses a modular Header and Footer, you cannot just double-click 'index.html'.
   Browsers block local file imports for security.
   
   SOLUTION:
   - Open this folder in VS Code.
   - Install the "Live Server" extension.
   - Right-click 'index.html' and choose "Open with Live Server".
